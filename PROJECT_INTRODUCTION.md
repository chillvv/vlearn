# Vlearn - AI 智能错题学习系统项目介绍（当前版本）

## 1. 项目概览

**Vlearn** 是一个以错题为核心的 AI 学习系统，目标是把“收集错题”升级为“结构化复盘 + 持续强化”的学习闭环。  
当前仓库以 **Web 前端 + Supabase 数据与函数层** 为主，已经覆盖从错题录入、分类归档、节点复盘到复习训练的主流程。

---

## 2. 当前技术架构

### 2.1 前端应用（`frontend/`）
- 技术栈：React 18、TypeScript、Vite、Tailwind CSS、shadcn/ui。
- 路由与页面：基于 React Router，包含登录、首页、错题库、草稿确认、专项练习、复习中心、设置等页面。
- 数据访问：通过 Supabase JS 进行鉴权与数据读写，同时保留 API 地址配置用于后端接口接入。

### 2.2 数据与后端能力（`supabase/`）
- 数据库：PostgreSQL（Supabase），围绕 `questions`、`user_weakness` 等核心表组织学习数据。
- 鉴权与安全：Supabase Auth + RLS 行级权限，按用户隔离数据。
- 服务端能力：包含 Supabase Edge Function（Deno/Hono）用于鉴权后接口与 AI 流式能力承接。

### 2.3 AI 能力
- 支持流式 AI 对话，并要求输出结构化错题卡片数据。
- 支持在“草稿确认页”和“节点复盘页”两类场景中使用 AI。
- 支持错题结构化字段抽取（题干、答案、解析、错因、方法等）。

---

## 3. 核心业务模块

### 3.1 账户与访问控制
- 邮箱登录与注册、登录态持久化、受保护路由跳转。
- 用户未登录时自动回到登录页，保障数据边界清晰。

### 3.2 错题录入与草稿确认
- 用户通过文本输入进入 AI 解析流程。
- 解析结果进入草稿确认环节后再写入正式错题库，降低脏数据进入概率。

### 3.3 错题库与节点化管理
- 错题支持按分类与节点聚合展示，并标记掌握度与顽固错题状态。
- 节点页集成“题目复盘 + AI 对话 + 关联内容”的一体化学习视图。

### 3.4 复习中心与专项训练
- 复习中心基于掌握度和复习状态组织待复习内容。
- 专项训练支持基于已有错题生成变式练习，形成针对性巩固流程。

### 3.5 设置与数据管理
- 提供账户信息管理与登出。
- 提供导入、导出、分享码等数据流通入口；其中部分能力已具备前端流程，后端对接仍在演进。

---

## 4. 数据流与学习闭环

1. **录入**：用户提交题目，AI 生成结构化草稿。  
2. **确认**：用户在草稿页确认后入库到 `questions`。  
3. **聚合**：系统按分类/节点组织错题并更新弱点统计。  
4. **复盘**：用户在错题与节点视图中复习，必要时调用 AI 辅助讲解。  
5. **强化**：进入专项训练，结果反向影响掌握度与后续复习优先级。  

---

## 5. 项目现状说明（便于对外同步）

- 当前可直接运行与演示的主体是 **Web 端（`frontend`）**。
- 数据层、安全策略与部分服务端能力落在 **Supabase（`supabase`）**。
- 仓库中存在历史文档与并行演进结构，介绍项目时建议以本文件与最新代码结构为准。

---

## 6. 快速上手（交接用）

1. 进入 `frontend/` 安装依赖：`npm i`  
2. 启动开发环境：`npm run dev`  
3. 根据环境配置连接 Supabase 与后端地址后即可进行联调。  

该文档用于项目交接时快速说明“系统做什么、现在做到哪、如何启动与演示”。
